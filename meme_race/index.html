What's the Problem?

This simulator models a "meme race" using two linear equations. Your goal is to find the solution to this system.

  • Use the "Time Machine" slider to move forward in time (days).
  • Watch how the slider value (t) changes the like counts in the Live Formulas.
  • Your challenge is to find the exact day t when both memes have the same number of likes.
  • Enter that day in the "Make Your Prediction" box and check your answer!

What do the numbers mean?

Rate (e.g., 50) = Growth Rate (new likes per day).

Time (t) (e.g., 0) = Time (the number of days passed).

Start (e.g., 300) = Starting Amount (likes on Day 0).
